# 🚀 CodeStream

**CodeStream** is a real-time collaborative coding platform that allows multiple users to write, edit, and review code together in a shared room. With built-in **audio/video conferencing**, **live code syncing**, and **room creation**, CodeStream is the ultimate tool for remote pair programming, technical interviews, group projects, and teaching.

---

## 🧠 Features

- 👥 **Room Creation & Management** – Create and join rooms to collaborate with teammates.
- 📡 **Real-time Code Syncing** – Seamless live syncing of code changes across all connected users.
- 🎥 **Audio & Video Conferencing** – Integrated communication via webcam and microphone.
- 📝 **Multiplayer Code Editor** – Syntax-highlighted editor supporting multiple programming languages.
- 🔐 **Secure & Scalable** – Built with privacy and performance in mind.
